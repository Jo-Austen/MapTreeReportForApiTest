1. List

1. List集合的概述和特点

  • 有序集合（序列），用户可以精确控制列表中每个元素的插入位置，可以通过整数索引访问元素
  • List中的集合元素可以重复（Set不可）
  • List中的元素有序，即存入和取出的元素顺序一样
public class List01 {
    public static void main(String[] args) {
        List<String> list = new ArrayList<String>();
        list.add("hello");
        list.add("world");
        list.add("future");
        //可重复
        list.add("hello");
        //输出元素有序
        System.out.println(list);

        //和Collection一样拥有Iterator迭代器方法
        Iterator<String> it = list.iterator();
        while (it.hasNext()){
            String s = it.next();
            System.out.println(s);
        }
    }
}

2. List集合的特有方法

  • Collection没有而List特有的方法（以下方法操作索引的时候需保证索引在范围之内，否则报错：IndexOutOfBoundsException）
  • void add(index,element)，没有返回值
  • E remove(index)，返回被删除的元素
  • E set(index,element)，返回被修改的元素
  • E get(index)，返回获取的元素
public class List02 {
    public static void main(String[] args) {
        List<String> list = new ArrayList<String>();
        list.add("hello");
        list.add("world");

        // void add(index,element)，没有返回值
        list.add(1,"hello");
        //E remove(index)，返回被删除的元素
        System.out.println(list.remove(0));
        // E set(index,element)，返回被修改的元素
        System.out.println(list.set(1,"word"));
        //E get(index)，返回获取的元素
        System.out.println(list.get(0));

        System.out.println(list);

    }
}

3. List.Iterator及其常用方法

  • 列表迭代器，继承Iterator，通过List对象的list.lterator()方法可以得到，是List集合特有的迭代器
  • 允许用户沿任一方向遍历列表
  • E next()，继承自Collection，返回迭代中的下一个元素
  • Boolean hasNext()：继承自Collection，下面还有元素，则返回True
  • E previous()和Boolean hasPrevious()，方法同next，但是方向相反
  • void add(E)：将指定的元素插入列表
  • 但是在使用迭代的时候还是直接使用iterator居多

4. List的遍历

public class List03 {
    public static void main(String[] args) {
        List<String> list = new ArrayList<String>();
        list.add("hello");
        list.add("world");
        list.add("future");

        System.out.println(list);

        //和Collection一样拥有Iterator迭代器方法
        Iterator<String> it = list.iterator();
        while (it.hasNext()){
            String s = it.next();
            System.out.println(s);
        }

        System.out.println("________________");

        //for循环+get
        for(int i =0;i<list.size();i++){
            String s = list.get(i);
            System.out.println(s);
        }

        System.out.println("________________");

        //listIterator的正向遍历
        ListIterator<String> lt = list.listIterator();
        while (lt.hasNext()){
            String s = lt.next();
            System.out.println(s);
        }
        System.out.println("________________");
        //listIterator的逆向遍历
        while (lt.hasPrevious()){
            String s = lt.previous();
            System.out.println(s);
        }

        System.out.println("________________");

        //增强for循环
        for (String s:list){
            System.out.println(s);
        }
    }
}

2. ArrayList和LinkedList

  • ArrayList底层是数组，查询快，增删慢
  • LinkedList底层是链表，查询慢，增删快
  • 两者的遍历方式同List
  • 两者共同拥有List和Collection的方法

LinkedList特有的方法

  • void addFirst(E)：在集合开头添加元素
  • void addLast(E)：在集合末尾添加元素
  • E getFirst()：返回集合中的第一个元素
  • E getLast()：返回集合中的最后一个元素
  • E removeFirst()：删除并返回集合中的第一个元素
  • E removeLast()：删除并返回集合中的最后一个元素
public class LinkedList01 {
    public static void main(String[] args) {
        LinkedList lkl = new LinkedList();
        lkl.add("hello");
        lkl.add("world");
        lkl.addFirst("first_add");
        lkl.addLast("last_add");
        System.out.println(lkl);

        System.out.println(lkl.removeFirst());
        System.out.println(lkl.removeLast());
        System.out.println(lkl);

        System.out.println(lkl.getFirst());
        System.out.println(lkl.getLast());
    }
}
