1. 集合概述

  • 多个数据，且数据个数可变时使用集合
  • 集合的特点：提供一种存储空间可变的存储模型，存储的数据容量可以发生变化
  • 内部存储对象，不能存储基本数据类型，除非将其转为包装类

1.1 集合体系结构

  • Collection是单列集合接口，Map是双列集合接口
  • List和Set都为Collection的子类接口，List中元素可重复，Set中的元素不可重复
  • 以上提到的均为接口，需通过实现类来调用其方法
  • Collection接口没有直接实现类，通过其子接口的实现类调用其方法（ArrayList等）

2. Collection

1. 概述

  • 是单列集合的顶层接口，表示一组对象，这些对象也称为Collection的元素
  • JDK不提供此类接口的任何实现（没有实现类），通过子接口（list和Set）的实现类调用其方法
  • 创建Collection集合对象的方法：通过多态方式由子接口的实现类创建，如ArrayList

2. collection集合常用方法

  1. Boolean add(element)：添加元素，输出结果永远为true
  2. Boolean remove(element): 移除元素，有则删除，并返回true，无则返回false
  3. void clear(): 清空集合中的元素，无返回值
  4. Boolean contains(element): 判断是否存在指定元素，存在返回true，不存在返回false
  5. Boolean isEmpty(): 判断是否为空，空为true，非空为false
  6. int size(): 返回长度
public class Collection01 {
    public static void main(String[] args) {
        //多态的方法创建，调用的方法都是对collection重写后的方法
        Collection<String> ct = new ArrayList<>();

        System.out.println("______test add______");
        ct.add("hello");
        //查看源码，该方法总是返回true
        System.out.println(ct.add("world"));
        //collection可以存储重复数据
        ct.add("world");
        System.out.println(ct);

        System.out.println("______test remove______");
        ct.remove("hello");
        System.out.println(ct.remove("hello"));
        System.out.println(ct);

        System.out.println("______test isEmpty______");
        System.out.println(ct.isEmpty());
        System.out.println(ct);

        System.out.println("______test contains______");
        System.out.println(ct.contains("hello"));
        System.out.println(ct);

        System.out.println("______test clear______");
        ct.clear();
        System.out.println(ct.isEmpty());
        System.out.println(ct);
        
        System.out.println("______test size______");
        System.out.println(ct.size());
        System.out.println(ct);
    }
}

3. collection集合的遍历

  1. Iterator<String> it = ct.iterator(); 通过方法创建迭代器，通过迭代器的方法遍历并返回元素
  2. boolean hasNext(); ：返回当前所在游标下面是否还有元素
  3. E next()：返回下一个元素
public class Collection02 {
    public static void main(String[] args) {

        //创建collection引用的ArrayList对象
        Collection ct = new ArrayList<>();
        ct.add("hello");
        ct.add("world");
        ct.add("world");


        Iterator<String> it = ct.iterator();

        /*
        ArrayList类内部的iterator方法，返回了一个Iterator<E>类型的Itr对象
        public Iterator<E> iterator() {
        return new Itr();
    }
    //Itr类调用了Iterator<E>接口
    private class Itr implements Iterator<E> {
        内部有next和hasNext方法
        ......
    }
    //将实现了Iterator<E>接口的类赋值给其实现类引用，相当于多态的形式
    //可以调用接口的重写方法
    //Iterator<E>接口内部也有next和hasNext方法
    Iterator<String> it = ct.iterator();
         */
        while(it.hasNext()){
            //System.out.println(it.next());
            //为了对下一个元素做操作，可以先将元素取出
            String s = it.next();
            System.out.println(s);
        }
    }
}

4. 增强for循环遍历

  • 简化数组和Collection集合的遍历，其内部原理是一个lterator迭代器
public class For00 {
    public static void main(String[] args) {
        /*
        for(元素类型 变量名:数组或collection集合){
            可直接使用变量名
        }
         */
        int[] arr = {1,2,5,4};
        for (int num:arr){
            System.out.println(num);
        }

        List<String> list = new ArrayList<>();
        list.add("hello");
        list.add("list");
        for(String s:list){
            System.out.println(s);
        }

        Collection<String> clt = new ArrayList<>();
        clt.add("hello");
        clt.add("collection");
        for (String s : clt){
            System.out.println(s);
        }
    }
}

6. 案例-存储学生对象的集合


//学生类
public class Student {
    int id;
    String name;

    public Student() {
    }
    public Student(int id, String name) {
        this.id = id;
        this.name = name;
    }

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public void show(){
        System.out.println("学号："+id+"，姓名："+name);
    }
}

//集合操作类
public class StudentCollection {
    public static void main(String[] args) {
        Collection<Student> ct = new ArrayList<Student>();
        ct.add(new Student(01,"lily"));
        ct.add(new Student(02,"lucy"));
        ct.add(new Student(03,"john"));

        Iterator<Student> it= ct.iterator();
        while (it.hasNext()){
            Student st = it.next();
            st.show();
        }
    }
}
