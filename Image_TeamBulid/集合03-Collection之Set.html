Set

1. Set集合的概述和特点

  • 不包含重复元素，没有带索引的方法，只能通过迭代器遍历
  • 其实现类内部元素部分有序，部分无序
  • 继承自collection，含有collection的方法
  • 是接口，不能直接实例化，可以用其实现类HashSet和TreeSet实现其方法
public class Set01 {
    public static void main(String[] args) {
        Set<String> set = new HashSet<String>();
        set.add("hello");
        set.add("world");
        set.add("java");

        System.out.println(set);
        // [world, java, hello]
        // HashSet内部元素的存储顺序和插入顺序不同，所以导致了迭代的顺序不同

        for (String s : set){//由于set接口没有索引，因此无法使用普通for循环，只能通过迭代器迭代
            System.out.println(s);
        }
        /*
        world
        java
        hello
         */

        set.add("hello");// 无法重复元素
        System.out.println(set);
        //[world, java, hello]

    }
}

2. 哈希值

  • 是JDK根据对象的地址或者字符串或者数字算出来的int类型的数值
  • 获取对象的哈希值：Object中的 public int hashCode()
  • 地址不同，则hashcode不同，但是可以通过重写方法改变这一特性
public class HashStudent {
    public static void main(String[] args) {
        Student student = new Student(15,"lily");
        System.out.println(student.hashCode());
        // 2054798982

        //新建了一个对象，将新的地址赋值给student，地址改变，hashcode改变
        student = new Student(16,"lala");
        System.out.println(student.hashCode());
        // 885951223

        //改变了对象的内容，但是地址值不变，所以hashcode不变
        student.setAge(25);
        student.setName("gogo");
        System.out.println(student.hashCode());
        // 885951223

        // 字符串在栈内存中，内容相同则地址相同，内容不同，地址不同
        System.out.println("hello".hashCode());// 99162322
        System.out.println("world".hashCode());// 113318802
        System.out.println("world".hashCode());// 113318802
        // 字符串重写hashcode方法，因此存在部分字符串的hashcode相同，但equals的内容不一定相同
        System.out.println("重地".hashCode()); // 1179395
        System.out.println("通话".hashCode()); // 1179395
    }

HashSet

1. HashSet的特点

  • 底层数据结构是哈希表
  • 不保证存储和取出元素的顺序一致，和Set一样，无法使用索引，也不包含重复元素
  • 默认长度是16，即0-15
public class HashSet01 {
    public static void main(String[] args) {
        HashSet hs = new HashSet();
        hs.add("hello");
        hs.add("world");
        hs.add("java");
        hs.add("hello");

        // 无序，且不重复
        System.out.println(hs);
        // [world, java, hello]
    }
}

LinkedHashSet

  • 哈希表和链表实现的Set结构
  • 哈希表保证了元素的唯一性（无法存储重复元素）
  • 链表保证的元素存入和取出顺序的一致性（有序）
  • 没有索引方法
public class LinkedHashSet01 {
    public static void main(String[] args) {
        LinkedHashSet<String> lhs = new LinkedHashSet<String>();
        lhs.add("hello");
        lhs.add("world");
        lhs.add("java");
        lhs.add("hello");

        System.out.println(lhs);
        //[hello, world, java]
        for (String str : lhs){
            System.out.println(str);
        }

        Set<String> set = new LinkedHashSet<String>();
        set.add("hello");
        set.add("world");
        set.add("java");
        set.add("hello");
        System.out.println(set);
       // ([hello, world, java])
    }
}

TreeSet

1. TreeSet概述

  • 没有带索引的方法
  • 不包含重复元素
  • 根据构造方法而按照一定规则排序
    • TreeSet()：无参构造根据元素的自然排序而排序
    • TreeSet(Comparator comparator)：根据指定的比较器进行排序
public class TreeSet01 {
    public static void main(String[] args) {
        //集合中不能存储基本数据类型，因此需存储包装类
        TreeSet<Integer> ts = new TreeSet<Integer>();
       //自动装箱转为包装类
        ts.add(1);
        ts.add(0);
        ts.add(52);
        ts.add(10);
        ts.add(10);//不包含重复元素

        System.out.println(ts);
        // 按照自然排序：[0, 1, 10, 52]
    }

3. Comparator接口

  • 该接口提供了compar(Type x,Type y)方法,用于比较两个对象的大小
  • 将该接口的实现类对象对象传递给TreeSet构造方法，使其内部元素按照定义的compar方法排序
  • 相比较Comparable接口接口更为灵活，不需要绑定到类
public class TreeSet03 {
    public static void main(String[] args) {
        TreeSet<Student03> ts = new TreeSet<Student03>(new Comparator<Student03>() {
            @Override
            public int compare(Student03 o1, Student03 o2) {
                int num1 = o1.getAge()-o2.getAge();
                int num2 = num1==0?o1.getName().compareTo(o2.getName()):num1;
                return num2;//内部定义排序的原则和自然排序定义相同
            }
        });

        Student03 s1 = new Student03(15,"lily");
        Student03 s2 = new Student03(15,"amy");
        Student03 s3 = new Student03(14,"zara");
        Student03 s4 = new Student03(15,"lily");
        ts.add(s1);
        ts.add(s2);
        ts.add(s3);
        ts.add(s4);
        for (Student03 s : ts){
            System.out.print("["+s.getAge()+","+s.getName()+"]");
        }
        //[14,zara][15,amy][15,lily]
    }
}

//按照总分从高到低排序，需后者减去前者
//需设置当成绩相同的情况下的解决办法，比如按照姓名排序，或者按语文，数学排序
public class TreeSet04 {
    public static void main(String[] args) {
        TreeSet<Student04> ts = new TreeSet<Student04>(new Comparator<Student04>() {
            @Override
            public int compare(Student04 o1, Student04 o2) {
                //看总分是否相同
                int num0 = o2.getChinese()+o2.getMath()-(o1.getChinese()+o1.getMath());
                //总分相同，num1=o2.getMath()-o1.getMath()
                //总分不同，num1=num0
                int num1 = num0==0?o2.getMath()-o1.getMath():num0;
                //总分不同，num=num0
                //总分相同，数学不同，num=o2.getMath()-o1.getMath()
                //总分相同，数学相同，num=o2.getName().compareTo(o1.getName())
                int num2 = num1==0?o2.getName().compareTo(o1.getName()):num1;
                //防止同名同姓，且成绩相同
                int num = num2==0?1:num2;
                return num;
            }
        });

        Student04 s1 = new Student04("lily",99,98);
        Student04 s2 = new Student04("mimi",89,76);
        Student04 s3 = new Student04("dingding",98,99);
        Student04 s4 = new Student04("dingding",98,99);
        ts.add(s1);
        ts.add(s2);
        ts.add(s3);
        ts.add(s4);

            System.out.println(s.getName()+","+s.getMath()+","+s.getChinese());
        }

    }
}
